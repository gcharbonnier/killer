Killer

Welcome to the Killer project.

Background technology

Killer backend is written in C++ whereas the front-end (UI) is written in QML, both supported by the Qt Framework. The design principle is a distributed Massive Role-Playing Pervasive Game : every players instance of killer communicate to a MySQL database that stores events between the killers instance. There is no centralized software in charge of the game logic.

How to start

  • download and install Qt Sdk.
    • Android :
      • download Android SDK and NDK, provide Qt creator with the according paths
  • fork the killer project on gitHub, build and deploy
